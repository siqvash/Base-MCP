# Base-MCP
Huge upgrade just dropped for AI agents on Base! 🤖  The Base MCP now supports 13 new "skills," expanding the protocol to 20 apps including OpenSea, Balancer, and KyberSwap.  This means agents can now transact, trade, lend, mint, and buy-all natively onchain, without leaving the chat interface
One standout tool, base-gas-mcp, even lets agents fetch real-time gas data for just $0.001 USDC via x402 payments.

But here's the best part: your private keys never leave your wallet-agents only prepare transactions, you always sign off.
